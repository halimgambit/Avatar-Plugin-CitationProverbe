# CitationProverbe
 
- AVATAR vous donne des citations ou proverbes aléatoire

- pas de regle fixe il suffit qu'il y a dans la phrase les mots citation ou proverbe

- exemple : une citation , je veux une citation , donne moi une citation, une autre citation, un proverbe ,je veux un proverbe , donne moi un proverbe, raconte un autre proverbe
